# create-gramstep

GramStep を Cloudflare にセットアップするための CLI です。

## Usage

```bash
npx create-gramstep@latest
```

明示的にコマンドを書く場合:

```bash
npx create-gramstep@latest setup
npx create-gramstep@latest setup --fresh
npx create-gramstep@latest redeploy
```

## Requirements

- Node.js 20+
- pnpm 10+
- Cloudflare account
- Meta Developer account
